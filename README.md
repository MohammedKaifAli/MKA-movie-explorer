# MKA-movie-explorer
# 🎬 Movie App

A simple and interactive Movie App built using **HTML, CSS, JavaScript**, and **JSON Server**.

---

## 🚀 Features

* 🎥 Browse all movies
* ❤️ Add to Favourites
* 🛒 Add to Cart
* ❌ Remove from Favourites/Cart
* 🔍 Search Movies
* 👤 User Login (LocalStorage)
* ⚡ Dynamic UI Updates

---

## 🛠️ Tech Stack

* HTML
* CSS (Flexbox)
* JavaScript (DOM, Fetch API)
* JSON Server (Fake Backend)

---

## 📂 Project Structure

```
movie-app/
│── index.html
│── favourite.html
│── cart.html
│── index.js
│── style.css
│── db.json
```

---

## ⚙️ Setup Instructions

1. Clone the repository:

```
git clone https://github.com/your-username/movie-app.git
```

2. Navigate to project folder:

```
cd movie-app
```

3. Start JSON Server:

```
npx json-server --watch db.json --port 3000
```

4. Open index.html using Live Server
